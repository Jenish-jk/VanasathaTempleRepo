## Run locally
1) Install Node.js (LTS)
2) In this folder:

```bash
npm install
npm run dev
```

Vite will print a local URL.

## Add temple photos
Place images in:
`src/assets/gallery/`

Rename them:
`1.jpg, 2.jpg, 3.jpg, 4.jpg` (you can add more)

Then update `src/data/templeData.js` if needed.

## Edit content
Open `src/data/templeData.js` and update timings, address, etc.
